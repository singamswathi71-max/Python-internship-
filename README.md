📊 Task 5: Data Analysis on CSV Files

📌 Objective

The objective of this project is to analyze sales data from a CSV file using Python and Pandas. The project demonstrates basic data analysis techniques and visualizes the results using a bar chart.

🛠️ Technologies Used

- Python
- Pandas
- Matplotlib
- Jupyter Notebook / Google Colab

📂 Project Structure

Task-5-Data-Analysis/
│── sales.csv
│── sales_analysis.py
│── sales_chart.png
└── README.md

🚀 Features

- Load data from a CSV file
- Display the first five rows of the dataset
- Check dataset information and shape
- Identify missing values
- Group data using "groupby()"
- Calculate total sales by product
- Generate and save a bar chart

▶️ How to Run

1. Clone this repository.
2. Install the required libraries:
   pip install pandas matplotlib
3. Place the "sales.csv" file in the project folder.
4. Run:
   python sales_analysis.py

📊 Output

- Displays dataset information in the console.
- Shows total sales grouped by product.
- Creates and saves a bar chart as "sales_chart.png".

📚 Learning Outcomes

- Reading CSV files with Pandas
- Working with DataFrames
- Using "groupby()" for data aggregation
- Handling missing values
- Creating visualizations with Matplotlib
